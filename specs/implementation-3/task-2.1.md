# Task 2.1: Create DigitalOcean Account

**Type:** Manual
**Performer:** User
**Phase:** 2 - Production Deployment

---

## What

Sign up for a DigitalOcean account and add payment method to enable Droplet creation.

---

## Objective

DigitalOcean will host the production Coolify instance on a Droplet (Virtual Machine) in the Sydney region. An account with valid payment method is required to provision infrastructure.

---

## How to Do

Visit the DigitalOcean website, create a new account or log in to an existing one. Complete email verification. Add a credit card or PayPal as payment method. Familiarize yourself with the DigitalOcean dashboard.

**Website:** https://www.digitalocean.com

---

## Expected Results

**Success Criteria:**
- DigitalOcean account created or accessed
- Email verified
- Payment method added and verified
- Can access DigitalOcean dashboard
- Account credentials documented securely

**How to Verify:**
- Log in to DigitalOcean dashboard successfully
- Dashboard shows "Ready to create your first Droplet" or similar
- Billing section shows payment method on file
